# RevoShop - E-Commerce Store

A modern e-commerce web application built with Next.js, featuring product browsing, detailed product views, and shopping cart functionality.

## 📋 Overview of the Project

RevoShop is a full-featured e-commerce store that allows users to browse and purchase tech products. The application provides a seamless shopping experience with a clean, responsive design and intuitive user interface. Users can explore products by category, view detailed product information, and manage their shopping cart with persistent storage.

## ✨ Features Implemented

### 🏠 **Home Page**
- **Hero Banner** - Eye-catching landing section with brand messaging
- **Category Carousel** - Interactive product category navigation
- **Featured Products Grid** - Curated product showcase with images, names, and prices
- **Responsive Design** - Optimized for desktop and mobile devices

### 🛍️ **Product Browsing**
- **Product Listing** - Complete catalog of available products
- **Search Functionality** - Find products by name with real-time filtering
- **Category Filtering** - Browse products by specific categories (audio, keyboard, mouse, storage, camera, smartphone)
- **Product Cards** - Clean product presentation with images, pricing, and quick actions

### 📱 **Product Detail Pages**
- **Dynamic Routing** - Individual pages for each product using Next.js dynamic routes
- **Detailed Information** - Complete product descriptions, specifications, and pricing
- **High-Quality Images** - Product photography with optimized loading
- **Add to Cart** - Direct product-to-cart functionality

### 🛒 **Shopping Cart**
- **Cart Management** - Add, remove, and clear cart items
- **Persistent Storage** - Cart data saved in localStorage across sessions
- **Real-time Updates** - Cart count updates instantly across all pages
- **Order Summary** - Total calculation and item count display
- **Checkout Simulation** - Complete purchase flow simulation

### 🧭 **Navigation & Layout**
- **Sticky Navigation** - Always accessible navbar with cart indicator
- **Client-side Routing** - Fast navigation without page reloads
- **Global Layout** - Consistent header and footer across all pages
- **Footer** - Brand information and additional links

### 🎨 **User Experience**
- **Responsive Design** - Works perfectly on all screen sizes
- **Loading States** - Smooth transitions and user feedback
- **Error Handling** - Graceful handling of missing products and edge cases
- **Modern UI** - Clean, professional design with Tailwind CSS

## 🛠️ Technologies Used

### **Frontend Framework**
- **Next.js 14** - React framework with App Router for modern web development
- **React 18** - Component-based UI library with hooks
- **TypeScript** - Type-safe JavaScript for better development experience

### **Styling & Design**
- **Tailwind CSS** - Utility-first CSS framework for responsive design
- **CSS Modules** - Scoped styling solution
- **Google Fonts (Geist)** - Modern typography

### **State Management**
- **Custom Hook Pattern** - Cart state management without Context API
- **localStorage** - Client-side data persistence
- **Event-driven Updates** - Real-time state synchronization across components

### **Routing & Navigation**
- **Next.js App Router** - File-based routing system
- **Dynamic Routes** - Parameterized URLs for product pages
- **Next.js Link** - Client-side navigation

### **Development Tools**
- **ESLint** - Code linting and quality enforcement
- **PostCSS** - CSS processing and optimization
- **npm** - Package management

### **Performance & Optimization**
- **Next.js Image** - Optimized image loading and rendering
- **Static Site Generation** - Pre-rendered pages for better performance
- **Code Splitting** - Automatic bundle optimization
